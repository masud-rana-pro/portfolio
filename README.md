# Md. Masud Rana Portfolio Website

This is a static, GitHub Pages-ready portfolio website generated from the CV data of Md. Masud Rana.

## Contents

- `index.html` - Main portfolio page
- `assets/css/style.css` - Responsive styling
- `assets/js/script.js` - Mobile navigation script
- `assets/img/profile.jpg` - Portfolio profile image
- `assets/cv/Md_Masud_Rana_CV.docx` - Downloadable CV
- `.nojekyll` - Keeps GitHub Pages simple and static

## How to publish on GitHub Pages

1. Create a new GitHub repository, for example: `masud-rana-portfolio`.
2. Upload all files and folders from this package to the repository root.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save and wait for GitHub Pages to publish.
6. Your portfolio URL will look like:
   `https://your-username.github.io/masud-rana-portfolio/`
